Why Speedy?

  • 100 % local

    Embeddings never leave your machine. Uses Ollama or any local model.

  • Incremental index

    Only re-indexes changed files. Daemon watches the filesystem in real time.

  • MCP compatible

    Plug directly into Claude, Cursor, or any MCP-aware editor.

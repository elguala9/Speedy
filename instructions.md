Functionality:

As a plugin of Opencode

Option:
 --daemons |-d -> let me see all the deamons in the pc
 --daemon-stop |-ds -> stop a daemon, it must pass the path with -p
 --daemon-restart | -dr -> restart a daemon after being stoppend, it must pass the path with -p
 --daemon-delete | -dd -> delete  a daemon forever, it must pass the path with -p
 --daemon-create | -dc -> create a daemon on a workspace if not exist, it must pass the path with -p
 --force | -f -> force the daemon to scan the all folder, it must pass the path with -p
 --workspaces | -w -> al the workspaces in the pc
 --workspace-create | -wc -> create a workspace and the daemon, it must pass the path with -p
 --workspace-delete | delete a workspace and the daemon, it must pass the path with -p

A daemon cannot exist withto workspace
